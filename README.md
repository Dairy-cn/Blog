# DBlog
DBlog是一款简洁美观、自适应的Java博客系统。使用springboot开发，前端使用Bootstrap。支持移动端自适应，配有完备的前台和后台管理功能。
网站预览    [https://www.zhyd.me](https://www.zhyd.me)

# 技术栈

- Springboot 1.5.9
- Apache Shiro 1.2.2
- Logback
- Redis
- Lombok
- Websocket
- MySQL
- Mybatis
- Mapper
- Pagehelper
- Freemarker
- jQuery 1.11.1
- Bootstrap 3.3.0
- jQuery Lazyload 1.9.7
- wangEditor
- fancybox
- iCheck
- 阿里云OSS
- kaptcha
